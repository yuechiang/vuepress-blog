# linux 安装 ftp 踩坑记

## 无法显示远程文件夹错误

![pic](/img/d58482fe725f29193b2e3ed547109963.png)

首先需要知道出现无法显示远程文件夹的原因是开启了被动模式，进去连接属性，点击选择，可以看到勾选了“使用被动模式”，如下图所示：

![pic](/img/8ed6c9bc4d51ad91f2efee6b6e8aef6c.png)
